# خطة دعم وتشغيل 3 سنوات (High level)

## SLAs (مقترح)
- **P1 حرجة (توقف كامل/اختراق/فقد بيانات)**: استجابة ≤ 1 ساعة، حل مؤقت ≤ 4 ساعات، حل دائم ≤ 48 ساعة.
- **P2 عالية**: استجابة ≤ 4 ساعات عمل، حل ≤ 5 أيام عمل.
- **P3 متوسطة/تحسينات**: تُجمع في إصدارات دورية شهرية/ربع سنوية.

## التشغيل والمراقبة
- مراقبة: Logs/Metrics/Traces + تنبيهات على (5xx، Latency، DB connections، Queue backlog).\n- لوحات: Availability, Error rate, Throughput, Auth failures, Integration failures.\n- إدارة الأسرار: تدوير كل 90 يوم (أو حسب سياسة الجامعة) + فصل صلاحيات.

## تحديثات أمنية
- تحديثات شهرية للتبعيات.\n- مراجعة OWASP Top 10 دوريًا.\n- اختبار اختراق سنوي (على الأقل) قبل تجديد.\n- فحص SAST على كل PR + فحص dependency vulnerabilities.

## النسخ الاحتياطي والاستعادة
- نسخة يومية + أسبوعية + شهرية.\n- اختبار استعادة ربع سنوي.\n- سياسات احتفاظ حسب NDMO وتصنيف البيانات.
