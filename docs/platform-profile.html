نظرة عامة على المنصة

Platform Overview

منصة اجتماعات جامعة حائل هي نظام إلكتروني متكامل صُمِّم خصيصاً لإدارة الاجتماعات واللجان الجامعية بكفاءة عالية. تغطي المنصة دورة حياة الاجتماع بالكامل من الجدولة إلى التقارير، مع دعم كامل للغة العربية واتجاه RTL، ونظام أدوار وصلاحيات متقدم يتكامل مع Azure Active Directory.

📚

إدارة شاملة للجان

إنشاء وإدارة لجان هرمية مع لجان فرعية، وتعيين الأعضاء والأدوار، مع دعم أنواع متعددة (تنفيذية، استشارية، فنية، خاصة).

📅

جدولة ذكية للاجتماعات

تقويم متكامل مع حجز القاعات، دعم الاجتماعات الحضورية والافتراضية (Teams, Zoom, WebEx)، وإرسال الدعوات تلقائياً.

📋

محاضر آلية للاجتماعات

إنشاء محاضر اجتماعات مع تسجيل الحضور والقرارات والتوصيات، ونظام اعتماد متعدد المراحل مع تصدير Word/PDF.

⚡

تواصل لحظي

محادثات فورية مع مؤشرات الكتابة، إشعارات فورية عبر WebPush، واستبيانات حية بنتائج لحظية.

إدارة اللجان

Committees Management

🏫 نظام شامل لإدارة اللجان

نظام متكامل لإنشاء وإدارة اللجان بهيكل هرمي يدعم اللجان الفرعية، مع تعيين الأعضاء وتحديد أدوارهم داخل كل لجنة.

أنواع اللجان

  • تنفيذية (Executive) — لجان صنع القرار العليا
  • إدارية (Administrative) — لجان الإدارة والتنظيم
  • استشارية (Advisory) — لجان الاستشارة والتوجيه
  • فنية (Technical) — لجان متخصصة
  • خاصة (Special) — لجان المهام المحددة

حالات اللجان

  • نشطة (Active) — معلّقة (On Hold) — مؤرشفة (Archived) — غير نشطة (Inactive)

الإمكانيات

  • إنشاء هيكل هرمي للجان مع لجان فرعية
  • تعيين رئيس اللجنة وأمين السر والأعضاء
  • عرض اجتماعات ومهام كل لجنة
  • إدارة طلبات التغيير على اللجان
  • عرض شجري للهيكل التنظيمي للجان (Hierarchy)

  • تسجيل الحضور — تسجيل حالة كل مدعو (حاضر، غائب، معتذر، متأخر)
  • محضر البنود — توثيق ما تم مناقشته في كل بند من جدول الأعمال

المهام والتوصيات

Tasks & Recommendations

☑ نظام متابعة المهام والتوصيات

تتبع المهام الناتجة عن الاجتماعات مع مهام فرعية ونسب إنجاز ومؤشرات أداء.

حالات المهام

  • معلّقة (Pending) — قيد التنفيذ (In Progress) — مكتملة (Completed)
  • متأخرة (Overdue) — ملغاة (Cancelled) — معلّقة مؤقتاً (On Hold)

مستويات الأولوية

  • منخفضة — متوسطة — عالية — حرجة

الإمكانيات

  • مهام هرمية: مهام رئيسية مع مهام فرعية (Subtasks)
  • تعيين المسؤولين وتحديد المواعيد النهائية
  • تتبع نسبة الإنجاز لكل مهمة ومهمة فرعية
  • لوحة تحليلات المهام (Task Dashboard)
  • عرض "مهامي" — المهام المسندة للمستخدم الحالي
  • ربط المهام بالاجتماعات واللجان

الاستبيانات والاستطلاعات

Surveys & Polls

📊 نظام استبيانات متقدم

إنشاء وإدارة استبيانات متنوعة مع تحليلات متقدمة وتصدير النتائج.

أنواع الأسئلة

  • اختيار واحد (Single Choice)
  • اختيار متعدد (Multiple Choice)
  • تقييم (Rating Scale)
  • نص مفتوح (Free Text)
  • مصفوفة (Matrix)

التحليلات

  • تحليل الإجابات مع رسوم بيانية تفاعلية
  • سحابة الكلمات (Word Cloud) للإجابات النصية
  • تحليل ديموغرافي للمستجيبين
  • تحليل زمني لمعدلات الاستجابة (Timeline)
  • تصدير النتائج إلى Excel
  • تصدير تقرير تحليلي إلى PDF
  • إرسال النتائج عبر البريد الإلكتروني

ميزات خاصة

  • قوالب استبيانات قابلة لإعادة الاستخدام
  • استبيانات مجهولة الهوية (Anonymous)
  • روابط مشاركة عامة
  • سحب عشوائي على المستجيبين (Lucky Draw)

الأدوار والصلاحيات

Roles & Permissions

الدور الوصف المسؤوليات الرئيسية
مدير النظام SystemAdmin إدارة كاملة للمنصة — المستخدمين، الأدوار، الصلاحيات، اللجان، الإعدادات
رئيس لجنة CommitteeHead رئاسة اللجنة، اعتماد المحاضر، إنشاء التقييمات، الإشراف على المهام
أمين لجنة CommitteeSecretary إنشاء الاجتماعات، كتابة المحاضر، إدارة الاستبيانات، رفع المرفقات
عضو لجنة CommitteeMember حضور الاجتماعات، التصويت، تنفيذ المهام، المشاركة في الاستبيانات
مراقب Observer متابعة ومراقبة سير الاجتماعات والقرارات بدون تدخل

النظام يعتمد نهج RBAC (Role-Based Access Control) مع صلاحيات تفصيلية على مستوى الإجراء. يدعم انتهاء صلاحية الأدوار (Role Expiration) وتعيين أدوار متعددة لنفس المستخدم. هناك ١٥+ فئة صلاحيات تشمل الإدارة، اللجان، الاجتماعات، المحاضر، المهام، والمزيد.

  • apps/web — تطبيق الويب (React + TypeScript + Vite)

الأمان والحماية

Security & Protection

🔒

مصادقة مزدوجة

دعم تسجيل الدخول عبر Azure AD (SSO) ومصادقة محلية بكلمة مرور مشفرة بـ BCrypt مع توكنات JWT.

🛡

تحكم بالوصول (RBAC)

نظام أدوار وصلاحيات تفصيلي مع دعم انتهاء صلاحية الأدوار وتدقيق الإجراءات.

📑

سجل تدقيق كامل

تسجيل كل إجراء حساس مع الطابع الزمني والمستخدم وعنوان IP لضمان المساءلة.

🗀

تصنيف الملفات

نظام تصنيف أمني للملفات (عام، داخلي، سري، سري للغاية) مع روابط تنزيل مؤقتة موقعة.

🔐

تشفير البيانات

تشفير كلمات المرور بخوارزمية BCrypt، وتوقيع التوكنات بـ HMAC-SHA256، مع دعم HTTPS كامل.

✅

إقرارات إلزامية

نظام إقرارات إجباري مع حجب الوصول (HTTP 451) حتى يتم التوقيع على الإقرارات المعلقة.
